#   Counting Bits


### 题意：
给一个非负整数num，遍历0<=i<=num的每个数字i，计算该数字的二进制表达中1的个数，并以数组形式将其返回。

#### Example 1:
Input: 2
Output: [0,1,1]

#### Example 2:
Input: 5
Output: [0,1,1,2,1,2]
