#	Longest Palindromic Substring
---

### 题意：
给一个字符串s，找出s中最长回文子串，可以假设s最长长度为1000

#### Example1:
Input: "babad"
Output: "bab"
Note: "aba" is also a valid answer.

#### Example2:
Input: "cbbd"
Output: "bb"
