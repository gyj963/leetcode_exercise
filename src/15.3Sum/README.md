#	3Sum
---

### 题意：
给一个数字数组 nums，nums中有没有三个数字a,b,c满足a+b+c=0？
找出数组中所有唯一的加合为0的三元组

#### Note:
结果集合中不能包含重复的三元组

#### Example:
Given array nums = [-1, 0, 1, 2, -1, -4],

A solution set is:
[
  [-1, 0, 1],
  [-1, -1, 2]
]

#### Testcase：
输入：
[0,0,0]
输出：
[0,0,0]
