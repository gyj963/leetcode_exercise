#	Longest Consecutive Sequence


### 题意：
给一个无序的数字数组，找到没出现在数组中最小的正数

算法时间复杂度要求：O(n) 空间复杂度 O(1)

#### Example1:
Input: [1,2,0]
Output: 3

#### Example2:
Input: [3,4,-1,1]
Output: 2

#### Example3:
Input: [7,8,9,11,12]
Output: 1
