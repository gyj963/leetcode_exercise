#	Coin Change


### 题意：
给一堆不同面额的硬币和总钱数。写一个函数计算你需要凑成总钱数所需的最少硬币数。如果总钱数不能通过任何数量的硬币来组成，返回-1

#### Example1:
Input: coins = [1, 2, 5], amount = 11
Output: 3 
Explanation: 11 = 5 + 5 + 1

#### Example2:
Input: coins = [2], amount = 3
Output: -1
