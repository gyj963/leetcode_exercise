#	Unique Binary Search Trees
---

### 题意：
给一个n，存储数字从1到n，结构上唯一的BST(二叉搜索树)的个数有多少？

#### Example:
Input: 3
Output: 5
Explanation:
Given n = 3, there are a total of 5 unique BST's:

   1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3

#### Testcase：
输入：
5
输出：
42

输入：
6
输出：
132
